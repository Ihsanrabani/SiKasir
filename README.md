# Overview

**SiKasir** is a website application that **helps waiters** record customer **orders quickly and in an organized manner.** This application is designed to **reduce order entry errors and speed up the service process in restaurants, cafes, or eateries.**

# Advantages
- Can be personalized with your business profile.
- Easy to understand and use
- Simple User Interface and User Experience

# Features
- Personalize App
- Add Orders
- Add Product
- Kitchen Display

# Tech Stack
**Front-End**
- React JS
- Vite
- Axios
- Tailwind CSS

**Back-End**
- Express JS
- Node JS

**Database**
- MySQL
